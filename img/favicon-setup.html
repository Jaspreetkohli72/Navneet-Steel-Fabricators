Steps to convert and use as favicon:

  1. Visit: https://convertio.co/image-converter/ or https://icoconvert.com/
  2. Upload the LinkedIn logo using the URL above or download it first
  3. Convert to ICO format (favicon format)
  4. Download the favicon.ico file
  5. Replace the existing favicon.ico in the img/ folder
  6. Clear browser cache (Ctrl+Shift+Del) to see the new favicon
